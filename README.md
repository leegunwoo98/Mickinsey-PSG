# Mickinsey-PSG solver
this is a solver for Mickensey-PSG game.

## How To
input the Mickensey-PSG data into the text field including food and food source. Preferably through Excel.
Next, try different combinations of animals to see if it works.

## Things to implement
this is not an automatic solver. You need to try different combinations. To implement an automatic solver, you would need the environment input, which would take too much time to input. Thus, it is not a good solution.
